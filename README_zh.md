Milk是Xepheris制作的UTAU虚拟歌手。本仓库是Milk英文Arpasing音源的镜像站。音频和oto均没有修改。

根据Milk的[用户协议](https://github.com/oxygen-dioxide/milk-eng/blob/main/license.md)，转载是允许的，但须正确地注明音源名称Milk和原作者Xepheris，并明确地声明你所作的改动。
